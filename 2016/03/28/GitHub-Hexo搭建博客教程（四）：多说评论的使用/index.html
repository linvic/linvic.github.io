GitHub+Hexo搭建博客教程（四）：多说评论的使用

  • GitHub
  • Hexo

我记得在第一篇Hexo系列博客就说到要给大家讲解如何使用多说评论进行。
其实很简单，那么今天就将给大家演示如何为咱们的博客添加多说评论。

添加多说评论功能

如果没有多说请先到多说进行注册。
登录多说，获取key，并将key添加到配置文件中
如下图：箭头所指的就是你的多说的key(注意:后面的.duoshuo.com不需要)

将你的key填写到E:Hexo/themes/yilia/_config.yml中

就是这么简单~
