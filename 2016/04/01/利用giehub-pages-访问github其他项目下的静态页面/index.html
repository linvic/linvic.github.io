利用giehub pages 访问github其他项目下的静态页面

  • GitHub
  • pages

本篇告诉你如何让我们的github项目实现在线预览。

我们都知道，咱们的项目push到github上面的时候，我们看到的是源代码，可是有时候我们想将我们项目的效果展示给他人时，就会想，如果github能够实时预览项目效果那该多好？

但是其实github pages可以解决我们这个问题。
先来讲下原理：咱们利用gh-pages分支，将需要预览的项目push到这个分支中，就可以实现在线预览项目效果了！

原理看懂了得大神应该已经明白了，下面的详细步骤适合还是没懂的朋友。

关于github pages的创建过程我这里就不复述了，我前面的博客搭建教程中有的。

本篇前提：

  • 拥有github pages

  1. 首先，我在github中新建了一个名为”demo”的项目，新建之后这里当然是没有内容的，咱们留着SSH备用（git@github.com:linvic/demo.git）

  2. 然后咱们新建一个文件夹（我在我电脑新建了一个E:demo）
