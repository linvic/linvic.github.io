GitHub+Hexo搭建博客教程之给博客设置个性域名.md

  • GitHub
  • Hexo

这个博客适合给想要给自己博客设置个性域名的人。
前提是你有购买域名。
这篇文章以阿里云万网域名为例。

博客设置

咱们先到我们博客这边设置一下我们要使用的域名。可以是顶级域名也可以是二级域名。我这里使用的是二级域名。
到E:\Hexo\themes\yilia\source目录下，新建一个名为CNAME（注意没有后缀名）的文件。内容如下：

PS：内容仅一行，就是你要设置的域名
这里再说明一下,为什么是在这个文件夹下面设置，而不可以是在public文件夹下设置，这是因为，我们每一次hexo g时，都是根据这个文件夹下的内容生成文件到public文件夹下面的。

域名服务商处设置

1.首先我们打开电脑的cmd命令提示符
然后键入命令ping linvic.github.io就可以看到博客ip地址了，记住这个地址，咱们要用。

  • 登陆阿里云管理控制台
  • 在我们的域名后面选择解析
  • 点击添加解析

    最后点击保存。
    然后我们就可以用blog.chenyanlin.com访问咱们的博客了。
