GitHub+Hexo搭建博客教程（一）：搭建前的准备工作

  • GitHub
  • Hexo

相信很多朋友们都和我一样，希望搭建一个自己的博客。
今天我就以我的经历，记录下利用github+hexo搭建博客的一些步骤。以及我遇到的一些坑的解决方法
此外，后面我还会写如何加入多说评论功能。

###为什么使用GitHub+Hexo搭建博客？
在我刚建立这个博客的时候我也说了，在我建成这个博客之前我碰到很多坑，最终选择了这个方法。Hexo是基于node.js快速、简洁、高效的博客框架，可以方便的在本地搭建。生成的静态页面托管到github pages上能够使我们很容易的就搭建成一套自己的博客。

###搭建前的准备工作
搭建前我们都应该做那些准备工作呢？如下：

  • 安装Node.js

  • 新建GitHub Pages

  • 安装GitHub客户端

  • github生成SSH公钥

如果存在的话，掠过下一步；不存在的请看下一步

  1. 生成SSH公钥

创建成功的话，你可以在自己的目录下，看到有一个.ssh目录

  1. 在github网站上面，为您的电脑添加公钥。在github官网上，点击右上角您头像的下拉选项，点击Settings。点击SSH keys选项，点击New SSH keys。C:\Users\LinVic\.ssh文件下的idrsa.pub文件使用记事本打开，复制里面的代码。添加到刚刚的网页输入框中。
记录这历史性的一刻，我的博客建立啦
